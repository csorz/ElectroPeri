UI Layout for QR Code Tool

How should the features be organized?

A

Tabs Layout

Three tabs: Generate | Decode | Batch. Each tab has its own form and options. Clean separation of features.

B

Unified Form

Single form with mode toggle at top. Style options always visible. Decode section collapsible at bottom. More compact.

C

Two-Column Layout

Left: Input form + options. Right: Preview + download. Style selector as visual picker. Decode as separate section below.
